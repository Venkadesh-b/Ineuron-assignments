1. What does the dollar($) sign do?

A. The dollar sign is used to make the refernce cell variable fixed, Where we need absolute reference not relative reference.

2. How to Change the Reference from Relative to Absolute (or Mixed)?

A. To use Absolute reference we need to use $ symbol in the reference variable. 
for example - $A$1 
Relative references are normal reference which requires the reference variable alone
for example - A1
Mixed reference is used when either column or the row is fixed.
for example - $A1 where column A is fixed and A$1 where row 1 is fixed.

3. Explain the order of operations in excel?

A. The excel follows the traditional math order of operation which is commonly known as BODMAS-> Brackets->Orders->Divisions->Multiplications->Addition->Subtraction.
